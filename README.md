## Selected Projects

### [Multimodal Clinical Reasoning](https://github.com/OxyOxygen/multimodal-clinical-reasoning)

Medical vision-language research project for generating radiology reports from chest X-ray images.

* BiomedCLIP vision encoder
* GPT-2 decoder with cross-attention
* Attention alignment regularization
* Ablation experiments across four model configurations
* BLEU, METEOR, ROUGE-L, CIDEr and clinical proxy evaluation

---

### [Customer Churn Prediction](https://github.com/OxyOxygen/Customer_Churn_Prediction)

End-to-end machine learning pipeline for customer churn probability estimation.

* Time-window-based feature engineering
* Customer transaction history aggregation
* CatBoost classification
* Stratified cross-validation
* Gini, Recall@10%, Lift@10% and ROC-AUC evaluation

---

### [Urban Mobility and Mood Analysis](https://github.com/OxyOxygen/urban-mobility-mood)

Statistical and machine learning analysis of environmental conditions, urban mobility and public mood in Istanbul.

* Multi-source dataset integration
* Hypothesis testing and correlation analysis
* Time-series feature engineering
* Regression and classification benchmarks
* Reproducible analytical workflow

---

### [AI Workstation Monitor](https://github.com/OxyOxygen/System_Monitor)

C++ system monitoring application designed for inspecting workstation performance during computational workloads.

* CPU, GPU and memory telemetry
* Session-based performance monitoring
* Diagnostic heuristics
* CSV and JSON export
* Modular hardware abstraction
